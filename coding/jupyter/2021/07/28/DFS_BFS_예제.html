음료수 얼려 먹기

  • N x M의 얼음 틀 존재
  • 구멍이 뚫려 있는 부분은 0 , 칸막이가 존재하는 부분은 1
  • 0으로 된 부분이 연결되어 있으면 1개의 아이스크림 덩어리로 간주
  • 인풋 첫번째는 N과 M
  • 두번째 부터 끝까지는 N x M의 얼음 틀의 상태를 전달
N,M = list(map(int,input().split()))
graph = []
for i in range(N) : 
    graph.append(list(map(int,input())))

def DFS(x,y ) : 
    if x<=-1 or x>=N  or y<=-1 or y>= M : 
        return False
    if graph[x][y] == 0 :
        # 방문 처리
        graph[x][y] = 1 
        DFS(x-1,y)
        DFS(x,y-1)
        DFS(x+1,y)
        DFS(x,y+1)
        return True # 한 블럭이 모두 방문처리되면 True 값 반환
    return False

result = 0 
for i in range(N) : 
    for j in range(M) : 
        if DFS(i,j)==True : 
            result += 1
print(result)

미로 탈출

  • N x M 크기 미로
  • 처음 위치는 (1,1), 미로 출구 위치는 (N,M)
  • 한번에 한칸씩 이동 가능
  • 괴물 있는 부분 0, 없는 부분 1
  • 탈출하기 위해 움직여야 하는 최소 칸의 개수
  • 칸을 셀 때는 시작 칸과 마지막 칸을 모두 포함해서 계산
from collections import deque
N,M = list(map(int,input().split()))
graph = []
for i in range(N) : 
    graph.append(list(map(int,input())))

# 이동할 네 방향 정의(상,하,좌,우)
dx = [-1,1,0,0]
dy = [0,0,-1,1]

# BFS 소스코드 구현
def bfs(x,y) : 
    queue = deque()
    queue.append((x,y))
    #큐가 빌 때 까지 반복
    while queue : 
        x,y = queue.popleft()
        for i in range(4) : 
            nx = x + dx[i]
            ny = y + dy[i]
            # 미로 찾기 공간을 벗어난 경우 무시
            if nx <0 or ny <0 or  nx>= N or ny>=M : 
                continue
            if graph[nx][ny] == 0 : 
                continue
            if graph[nx][ny] == 1 : 
                graph[nx][ny] = graph[x][y]+ 1 # 이동 하기 전 위치에 1을 더해줌
                queue.append((nx,ny))
    return graph[N-1][M-1] # 마지막 위치에서의 이동 횟수를 반환 

print(bfs(0,0))
